# 7. 프로젝트 기획 의도

- 본 프로젝트는 1919년 서울 3.1운동의 준비 배경과 치밀했던 과정, 그리고 뜨거웠던 당일의 시위 장소를 조명합니다. 서울 거리 곳곳에 남겨진 치열했던 항쟁의 발자취와 3.1운동가들을 텍스트 위주에서 벗어나 역사 자료를 시각화하여, 스크롤 형식으로 자연스럽게 전개하여 100여 년 전 그날의 현장을 직관적이고 생생하게 따라갈 수 있도록 기획했습니다.

# 12.  기대효과

- 교육 및 학술적 측면
    - 입체적인 역사 학습 경험 제공: 텍스트 중심의 기존 역사 교육에서 벗어나, 지리적 정보(지도)와 서사(스토리)를 결합하여 사용자의 직관적인 이해를 돕습니다.
    - 지역 기반의 독립운동사 재조명: 널리 알려진 주요 사건 외에도, 사용자 주변이나 특정 지역에서 일어난 숨겨진 만세 운동의 역사와 지역 유공자를 발굴하고 알릴 수 있습니다.
    - 올바른 역사 의식 고취: 3.1운동의 역사적 현장을 시각적으로 체감하게 함으로써, 주 이용층(학생, 일반 시민 등)의 역사적 자긍심과 보훈 의식을 높일 수 있습니다.
- 관광 측면
    - 역사 테마 관광 활성화: 스토리맵에 표시된 경로를 따라 직접 현장을 방문하는 '도보 탐방 코스'나 '스탬프 투어', ‘Qr 투어’ 등의 관광 상품으로 연계할 수 있습니다.
- 기술 및 서비스 측면
    - 분산된 역사 데이터의 통합 및 시각화: 여러 문헌이나 기관에 흩어져 있던 3.1운동 관련 사료, 사진, 위치 정보 등을 하나의 플랫폼에 모아 의미 있는 스토리맵으로 구축합니다.

# 14. 핵심 기능

- 핵심 기능1
    - 패럴랙스 스크롤링 기법을 적용하여 사용자의 스크롤에 따라 콘텐츠가 변화하도록 구현하였습니다. 이를 통해 단순히 정보를 읽는 방식에서 벗어나 사용자가 직접 탐색하며 역사 콘텐츠를 경험할 수 있도록 하였습니다.
    - 패럴랙스 스크롤링 : 사용자의 스크롤에 따라 콘텐츠가 순차적으로 나타나는 웹 구현 기법정보 전달과 사용자 만족도 향상에 긍정적인 효과가 있는 것으로 나타났다.
    - 본 프로젝트는 이러한 특징을 활용하여 3·1운동의 전개 과정과 주요 유적지를 하나의 흐름으로 연결하고, 사용자가 사건의 진행 과정을 자연스럽게 탐색할 수 있도록 구성하였다.
    - 관련논문 / 인용문구
        - 정보습득이 이용의도에 가장 큰 영향을 주었고 (β=.455), 다음은 유쾌성 (β=.285), 심미성 (β=.230) 순으로 나타났다.
        - 기존 영상 콘텐츠는 사용자들이 선택할 경우에 한하여 일방적으로 정보를 전달하는 반면 패럴랙스 스크롤링 기법은 웹사이트를 보는 사용자의 스크롤 속도에 따라 인터랙
        티브하게 영상을 재생하는 듯한 효과를 제공함으로써 사용자와 쌍방향 커뮤니케이션이 가능하다.
        - 참고문헌 : 장기원·윤재영, 「패럴랙스 스크롤링 웹사이트의 정보습득이 이용의도에 미치는 영향」, 커뮤니케이션디자인학연구 제79호, 2022.
            

- 핵심기능2
    - 지도 데이터 시각화 기법을 적용하여 사용자가 3·1운동의 지리적 확산 과정을 직관적으로 파악할 수 있도록 구현하였습니다. 이를 통해 텍스트 위주의 단편적인 정보 제공에서 벗어나, 역사적 사건이 일어난 공간적 맥락을 입체적으로 경험할 수 있도록 하였습니다.
    - 지도 전달 기법 : 복잡한 지리적 정보나 사건의 발생 위치를 지도 상에 시각적으로 맵핑하여 전달하는 웹 구현 기법. 정보의 직관적인 인지와 공간적 이해도를 높이는 데 긍정적인 효과가 있다.
    - 프로젝트 적용 : 본 프로젝트는 이러한 특징을 활용하여 section 1, 2, 4에 걸쳐 만세운동의 발생 지역과 확산 경로를 시각화하고, 사용자가 3·1운동의 전국적 규모와 흐름을 거시적인 관점에서 자연스럽게 탐색할 수 있도록 구성하였다.
    - 기존 방식과의 비교 : 기존 텍스트나 표 위주의 역사 콘텐츠는 사용자가 사건의 지리적 규모와 동선을 단번에 파악하기 어려운 반면, 지도 전달 기법은 웹사이트를 보는 사용자에게 사건의 발생 위치와 확산 경로를 공간적 맥락과 함께 제공함으로써 정보 습득의 효율성과 역사적 몰입감을 극대화한다.

- 핵심기능3
    - 지도 상의 주요 역사적 거점에 펄스(Pulse) 애니메이션 효과를 적용하여 사용자의 시선을 자연스럽게 집중시키도록 구현하였습니다. 이를 통해 정적인 지도에 생동감을 불어넣고, 3·1운동의 동시다발적인 확산세를 역동적으로 경험할 수 있도록 하였습니다.
    - 지도 펄스 효과 : 특정 마커나 위치 위로 파동이 퍼져나가는 듯한 시각적 애니메이션을 제공하는 기법. 웹사이트 내 시각적 계층 구조를 명확히 하고 사용자의 주목도를 즉각적으로 높이는 데 효과적이다.
    - 프로젝트 적용 : 본 프로젝트는 이러한 특징을 활용하여 section 3의 주요 지도 영역에 펄스 효과를 적용하고, 전국적으로 거세게 뻗어나갔던 만세운동의 뜨거운 열기와 역사적 현장의 생동감을 마치 심장 박동처럼 직관적으로 표현하도록 구성하였다.
    - 기존 방식과의 비교 : 기존의 정적인 지도 마커는 사용자의 능동적인 시선을 유도하고 사건의 에너지를 전달하는 데 한계가 있는 반면, 펄스 효과는 사용자에게 즉각적이고 동적인 시각적 피드백을 제공함으로써 가장 중요한 역사적 거점 정보에 대한 주목도를 높이고 웹사이트의 심미적 만족도를 향상시킨다.
